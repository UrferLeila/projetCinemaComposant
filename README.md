# Projet Cinéma

## Description

Voici notre projet simulant un cinéma fictif : Le Palace. 
Permettant de réserver des sièges, en fonction des séances des films.
Elle permet aussi une gestion admin.

## Prérequis

Installer node-js
Installer wampserer
Installer sur le site de Wamp, les VC++ Packages
(a mettre en ficher annexe ficher all in one du partage + 2015 version)
mettre php 08.04.15 dans le variable denvironement
copier le .env.dev les coller et le renomer .env
installer composer
composer install 
npm install
npm install vue axios
npm install @vitejs/plugin-vue
(lancer les deux serve)
faire la base de donné (wipe, migrate, seed)

## Installation

1. Cloner le dépôt :

   ```bash
   git clone <url-du-repository>
   ```
2. Accéder au dossier du projet :

   ```bash
   cd projet-cinema
   ```
3. Installer les dépendances (si nécessaire)
4. Lancer l’application selon la configuration choisie
